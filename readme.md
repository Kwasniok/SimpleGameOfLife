# Simple Game of Life

A functional approach to Conway's game of life.

## Run
Start via `python main.py` and advance the simulation with `Enter`. Abort the simulation with `Ctrl+C`.

## Requirements
- [Python 3.6+](https://www.python.org/downloads/)
